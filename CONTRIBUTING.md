# Contributing to Echo

We're glad you're interested in contributing to **Echo**.

Your contributions help make the platform better for everyone.

### Get Started

Check out the project README and codebase to get familiar with the architecture. For questions or coordination, reach out to [Facile Studio](https://facile.studio).

### Code Quality

Before sending us your code, double-check that it meets our coding standards. You can do this by running: `npm run lint`. If there are any issues, you can fix them by running: `npm run lint-fix`. Once your code passes these checks, feel free to submit your pull request.

**Happy coding!**
